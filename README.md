# VISION OCR

## Overview

### Installation

1. Clone source code

```bash
$ git@github.com:loyal-ki/vision-ocr.git
$ cd vision-ocr
```

2. Initial setup && build the Docker image

```bash
$ make initialize
```

3. Run source

```bash
$ make up
```

### Swagger

- Local: http://localhost:5678/docs

- Docker: http://localhost:8082/docs
